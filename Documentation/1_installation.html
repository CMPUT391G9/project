1. Installation Guideline

To install the system, you need to do the following steps:

1.1 Download the source files

  • Download the tar file.

  • Extract source code into the proj1 directory within your catalina/webapps.

  • Copy the GetOnePic.java file into catalina/webapps/WEB-INF/classes directory.

  • Move the commons-fileupload-1.0.jar file into catalina/webapps/WEB-INF/lib directory.

1.2 Run Tomcat

  • Open a terminal, and run starttomcat

1.3 Set up the initial database

  • Start sqlplus in a termial and login.

  • Set up and populate the initial database using script files (e.g. setup.sql) by running the following commands:
    @setup.sql


1.4 Connect to Oracle DB

  • You need to connect to Oracle DataBase at http://HostName:PORT/proj/Connector.html where HostName is the machine name of the server the system installed on and PORT is the port number that tomcat is running on.

  • Enter the Oracle ID and Password, and click "CONNECT" button
